# global_reading
существует метод глобального чтения для преодоления любых сложностей с этим навыком. Этот подход заключается в том, что человек запоминает и узнает слово целиком по зрительному образу, без подробного буквенного разбора. Людям с дислексией сложно прочитывать каждый слог, поэтому при коррекции применяется способ глобального чтения.
