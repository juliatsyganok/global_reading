# global_reading
## Суть: 
существует метод глобального чтения для преодоления любых сложностей с этим навыком. Этот подход заключается в том, что человек запоминает и узнает слово целиком по зрительному образу, без подробного буквенного разбора. Людям с дислексией сложно прочитывать каждый слог, поэтому при коррекции применяется способ глобального чтения. Эффективность данного подхода подтверждена в научных исследованиях. В случае дислексии невозможно добиться улучшений без комплексной коррекции, поэтому этот проект будет направлен именно на отслеживание и фиксацию статистики улучшений в течении коррекционных занятий. 

## Идея:
Создать сервис, который будет сохранять и отслеживать статистику улучшений у пользователя с дислексией. Ключевая идея в том, чтобы взять за основу подход глобального чтения через анализ синхронизации курсора и голоса при чтении. Приложение отслеживает положение курсора и произнесенное слово (слог) в конкретный момент времени. Таким образом мы можем отслеживать, как человек воспринимает информацию: читает слово по слогам или угадывает его по визуальному образу и контексту. При наблюдении прогресса, получим некоторое отставание курсора от голоса в рамках одного слова. В статистику необходимо также добавить процент слов, прочитанных с первой попытки, скорость чтения. Если углубиться в теорию, также можно отслеживать преобладающую точку фиксации взгляда, это показывает, как быстро человек понимает слово. Таким образом мы используем технологии машинного обучения, а именно компьютерное зрение для курсора и анализ речи для аудио. 

## Описание сервиса:
Пользователю предлагается текст. Необходимо записать его аудио прочтение и видеозапись экрана с движением курсора.
Полученные файлы отправляются в тг-бот. С помощью ML анализируются два потока информации с привязкой к времени.
Пользователь получает статистику улучшений


Проблема: Сложности отслеживания прогресса при коррекции дислексии
Решение: Анализ синхронизации курсора и голоса при чтении через ML
Актуальность: Научно доказанная эффективность глобального чтения при дислексии
Аналоги: трекинги скорочтения, трекинги коррекции дислексии
